About Lock Screen:

The lock screen is the screen which opens when you hit the project URL.

Here the current time and date is displayed on the image.

A click on the image takes you to the login screen.

The main idea of this screen is to resemble the Microsoft Windows lock screen.

Screen shots

A message appears at the top right corner to click the screen for unlocking.

After 4 seconds, the message disappears and the screen would be as below:

Implementation:

The time and date are calculated in a controller specified to lock screen called LockScreenCtrl.

The current date and time are achived using Date object.
