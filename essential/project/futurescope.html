Future Scope:

I developed this project with an aim of learning as much as possible. I especially concentrated on AngularJs, NodeJs and MongoDB.
Apart from the MEAN stack technologies, I also learnt HTML5, CSS, Javascript and JQuery.

After completing the project, I gained a proficient knowledge in AngularJS.

This project can be extended into a larger scale by adding widgets related to any perticular field.

  • It can be implemented into an online desktop whcich includes file storage and file upload.
  • This can be modified into a portal for an organization where tabs are replaced by widgets. In this way, users can access multiple tabs at the same time.

I really like the idea of an online desktop and would love to extend this project by adding more features.
