About Data Storage in MongoDB:

All the data in this project is stored in a NoSQL database MongoDB. The connectivity between the MongoDB and server is done using Mongoose.
The following are the collections in my project:

Collection Description
Authentication This collection has 2 fields. Username and password. The ID of the user assigned here will be used as the unique identifier throught the application.
UserProfile This collection has 10 fields, which includes the user details and the unique identifier of the user(from the Authentication collection).
Message
This collection all the messages between the users. It has 3 fields (From, To, and Message).
The from and to details are stored as the unique identifiers of the users.
WidgetData
This collection has 14 fields including the unique user ID field and an object for every widget.
Each user has an entry in this table and stores the position and other required information of widgets.
