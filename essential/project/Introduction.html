Introduction

Widgets is a single page web application project developed as a part of CS5610 - Web Developement course that I took in my graduation. I developed this project to display my level of knowledge and understanding in AngularJS, NodeJS and MongoDB.

This content management system allows users to access many different usefull widgets in a single page. Each widget can be arranged at desired size and position on the screen.

List of widgets:

Name Description
Google Search This widget allows users to do a google search using some text.
Notepad Notepad widget alows users to create instant notes. Multiple notes can be created and saved.
Browser A user can access different websites from the browser widget. Any number of tabs can be created.
Youtube The youtube widget allows users to search youtube videos, play them and also add them as a favorite.
Narrator This widget reads out any english text entered.
Calculator Calculator is a widget which allows users to perform basic mathematic operations.
Messenger This widget allows users to communicate(via text messages) with any other user registered into the system.
Weather This weather widget allows users to know the currect weather(including the min and max temperatures of the day).
Stop Watch Stop watch is a widget which allows users to measure the amount of time elapsed. It also allows mainitain lap time.
E-Mail This allows users to send an email to others. This widget also includes the feature of saving as a draft.
Data and Time This widget is alwas to the right most of the taskbar. It displays current time and date. When mouse overed, it displays the day name.

Different pages in application:

Name Description
Lock Screen This is the screen which gets loaded on hitting the project URL. Here an image is displayed with current time and date. When clicked anywhere on the page, it redirects to login page. The idea is to resemble, Microsoft windows lock screen.
Login Screen The Login screen features different features other than just user login, like admin login, registration, recover password, and navigation link to use as a free user.
Home Screen The screen where all the widgets are accessed. A free user and registered user come to the same page.
Admin Screen This screen can be accessed only by the admin. It contains all the admin options to manage the data.

Techonologies Used:

Technology Used at Description
HTML Client Side All the screens were develoepd using HTML
CSS Client Side Styling for the screens and widgets are done using CSS
Java Script Client Side Used for validations and many string handling functions.
JQuery Client Side This is used when ever AngularJS jQLite fails to work. Used especially for resizing feature of the widgets.
jQuery-UI Client Side Used especially for resizing feature of the widgets.
AngularJS Client Side This is most extensively used technology in project. Used to handle data and navigations.
jQLite Client Side This AngularJs's version of JQuery, is used in creating many UI effects.
NodeJs Server Side This technology is the backbone of server side functionality.
ExpressJS Server Side This technology is a module of NodeJS. It is used in establishing web services required for the project.
PassportJS Server Side This technology is a module of NodeJS. It is used for handling sessions and authentication of users.
MongoDB Data Storage This noSQL database is used as the backend of my project.
Mongoose Server Side This NodeJs module helps in integrating MongoDB at server side.
Node Mailer Server Side This NodeJs module helps in sending mails.
