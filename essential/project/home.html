About Home Screen:

The screen where all the widgets are accessed. A free user and registered user come to the same page.

But a free user has only access to some of the widgets.

The below is the table showing the list of available widgets and the accesability.

Name Accessibility Description
Google Search Free & Registered Users This widget allows users to do a google search using some text.
Notepad Free & Registered Users
Notepad widget alows users to create instant notes. Multiple notes can be created and saved.
Free User - All the added notes are temoporary.
Registered User - All the added notes are saved.
Browser Free & Registered Users
A user can access different websites from the browser widget. Any number of tabs can be created.
Free User - All the changes are temoporary.
Registered User - All the changes are saved.
Youtube Free & Registered Users
The youtube widget allows users to search youtube videos, play them and also add them as a favorite.
Free User - All the favorites are temoporary.
Registered User - All the favorites are saved.
Narrator Free & Registered Users
This widget reads out any english text entered.
Calculator Free & Registered Users Calculator is a widget which allows users to perform basic mathematic operations.
Setting Free & Registered Users
This setting widget allows to change the background image, user profile and change password.
Free User - Only background changing tab is accessible.
Registered User - Can chanbe background, edit user info and change password.
Messenger Registered Users This widget allows users to communicate(via text messages) with any other user registered into the system.
Weather Free & Registered Users This weather widget allows users to know the currect weather(including the min and max temperatures of the day).
Stop Watch Free & Registered Users Stop watch is a widget which allows users to measure the amount of time elapsed. It also allows mainitain lap time.
E-Mail Registered Users
This allows users to send an email to others. This widget also includes the feature of saving as a draft.
Accessible only if the user verifies his Email ID.
Save Registered Users This widget helps in saving the current position of the widgets.
Exit Free & Registered Users
Free User - Exits the system and navigates to login screen.
Registered User - Logs off the user and navigates to login page.
Hide All Widgets Free & Registered Users This widget hides all the showing widgets.
Data and Time Free & Registered Users This widget is alwas to the right most of the taskbar. It displays current time and date. When mouse overed, it displays the day name.

Implementation:

All the current states are saved in a UserService. when an action is performed, the controller calls a function in the service and performs the action.

Some actions require a DB interaction. Such tasks are performed by asking for the data from the server through the service. On receiving the data, a callback of the controller is executed to display the data on the view.
