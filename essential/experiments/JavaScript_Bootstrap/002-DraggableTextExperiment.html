Draggable Text

  • For making a text draggable, the following steps must be followed

    • Give Ids tothe draggable and dragging into elements.
    • For elements which are for dropping on, call java script functions for events "ondrop" and "ondragover"
    • For elements which are dragged, call java script functions for events "draggable" and "ondragstart"

    In the javascript, write the below code:

    function allowDrop(ev) {
    ev.preventDefault();
    }
    function drag(ev) {
    ev.dataTransfer.setData("object", ev.target.id);
    }
    function drop(ev) {
    ev.preventDefault();
    var data = ev.dataTransfer.getData("object");
    ev.target.appendChild(document.getElementById(data));
    }
