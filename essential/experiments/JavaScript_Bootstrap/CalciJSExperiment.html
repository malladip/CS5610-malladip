Java Script

  • Linking a External JavaScript file into the HTML File

    <script type ="text/javascript" src="CalciJS.js"></script>

  • Accessing a HTML form element in the JavaScript file using ID.

    document.getElementById('output_display').value;

    Similarly we can access using class name, and form name.

  • Evaluating a string expression.

    A mathematical expression in string format can easily be evaluated by simply passing the string as an argument to the function eval()

    eval(equation);

  • Try Catch Statements in JavaScript

    As in Java, we can have exception in java script. The syntax is simillar to Java.

    try {
    document.getElementById('output_display').value = eval(equation);
    } catch (e) {
    alert("Message");
    }
