CSS Responsive Design

  • In this experiment, I created a responsive design of a page. Here I created 3 different CSS files for desktop, tablet and mobile device (one for each). I used the media tag to achive the condition check.

    For Desktop version:
    @media screen and (min-width: 1235px){ }

    For Tablet version:
    @media screen and (min-width: 835px) and (max-width: 1235px){ }

    For Mobile version:
    @media screen and (min-width: 328px) and (max-width: 835px){ }
