In this experiment, I tried to embed a youtube video into the current page.

  • All youtube videos have a URL of perticualr format.

    https://www.youtube.com/watch?v=eTUstNknCrw

    To embed, all we need is the code after the "=" sign in the URL

    A youtube video is embeded as an IFrame

    <iframe width="1280" height="720" src="https://www.youtube.com/embed/eTUstNknCrw" frameborder="0" allowfullscreen></iframe>

    So we can edit our required attributes of the iframe and customize the embedding video

  • Creating a new element using JQuery

    var new_iframe = $("<iframe />")
    .attr('class', 'video thumbnail')
    .attr('src', new_link)
    .attr('frameborder', '0')
    .css('float', 'left')
    .appendTo('#exp_container');
