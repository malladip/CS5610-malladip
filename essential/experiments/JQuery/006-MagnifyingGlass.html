Magnifying Glass

This experiment is about magniying glass which is used in many shopping websites.

This experiment uses CSS3 and jQuery.

$("#go").click(function () {
    var url = $("#url").val();
    var image_background = "url('"+ url + "') no-repeat";
            
    $("#image").attr('src', url);
    $(".large").css('background', image_background);
    
});
