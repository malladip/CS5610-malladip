In this experiment, I extended the previous experiment by making the vides as draggable and resizable.

  • This experiment needs JQuery UI. This can be linked by below code.

    <link rel="stylesheet" href="css/jquery-ui.css" />
    <script src="js/jquery.js"></script>
    <script src="js/jquery-ui.js"></script>

  • After adding the JQuery UI plugin, many useful functions such as draggable, resizable and sortable etc can be used.

    var draggable_div = $("<div>")
    .attr('class', 'video thumbnail')
    .resizable()
    .draggable()
    .attr('draggable','true')
    .appendTo('#exp_container');
