This experiment is my Hello World in JQuery.

Here I tried to display the current time(local time of the user).

  • For linking the jQuery framework,the below code must be in the head tag.

    <script src="//code.jquery.com/jquery-1.11.2.min.js"></script>
    <script src="//code.jquery.com/jquery-migrate-1.2.1.min.js"></script>

  • For letting the DOM load before the JQuery function starts, we first call a function in which we again call actual functions.

    $(function () {
    ...our code goes here...
    })

  • JQuery has a powerful way of selecting elements.
    # for ID
    . for Class
    Similarly, many follows.

    $("#date").find("#month").html(month + 1);

  • We can make a part of code execute after a specfied interval of time as below.

    setInterval(function () { }, 1000);
