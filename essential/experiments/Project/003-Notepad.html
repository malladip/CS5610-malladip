In this experiment, I created a notepad widget where we can store instatnt notes.

The following are the features of the notepad:

  • Can add new note by clicking on the + sign.
  • Can change the view of the notepads into a list form by clicking on the second button.
  • Can change the view of the notepads back into a grid form by clicking on the grid ikon button.
  • Click on any note to see the entire note.
  • On clicking back button, the note gets atomatically saved.
  • Can delete a note by clicking on the delete ikon button
