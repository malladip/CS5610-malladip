Toolbar:

In this exeriment, I made a toolbar.

This toolbar has buttons which are sortable. i.e change the order of buttons on the toolbar.

$('#sortable').sortable({
        handle: 'button',
        cancel: ''
    })

And the toolbar position can be changed, by dragging it either to the top, right, left or bottom.

if (currentY < 55) {  // Top
toolbar
.removeClass("toolbar-left")
.removeClass("toolbar-right")
.removeClass("toolbar-bottom")
.addClass("toolbar-top");
}
