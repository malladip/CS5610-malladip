In this experiment, I created a I tried a create a windows lock screen and login page. This is how my home page would look like in my project.

In this process, I learnt the use of the css unit 'vh'

#LockScreen {
    background-image: url("../images/lamborghini-desktop-440x900.jpg");
    height: 100vh;
    font-family: Arial;
    color: white;
}

The first image wcich appears on the screen is the lock screen. By clicking anywhere on the image, will redirect to the login page.

$('#LockScreen').click(function () {
        $location.url("/login");
    });

The lock screen also provides with the current time of the user. It changes dynamically and refreshes every second.

 $interval(function () {
        dateObj = new Date();
        $scope.time = getTime(dateObj);
        $scope.date = getDate(dateObj);
    }, 1000);

In the lock screen, the lock button at the bottom left corner is used to navigate back to the lock screen.

 $scope.lock = function () {
        $location.url("/lock");
    }
