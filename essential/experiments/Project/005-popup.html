Dialog Box In AngularJS

In this experiment, I reaserched on how to create a popup dialog box using angularJS.

This involved lot of effort as there was no good documentation for this.

Finally I came to know that, we can use a directive called ngDialog which allows us to create a dialog box

Click here to see the output of a dialogboox using angularjs


This experiment involves, embeding the following scripts:

<script src="https://cdnjs.cloudflare.com/ajax/libs/ng-dialog/0.3.12/js/ngDialog.min.js"></script>
<link href="https://cdnjs.cloudflare.com/ajax/libs/ng-dialog/0.3.12/css/ngDialog.min.css" rel="stylesheet" />
<link href="//cdnjs.cloudflare.com/ajax/libs/ng-dialog/0.1.6/ng-dialog-theme-default.min.css" rel="stylesheet" />

The below is the code for opening the dialog:

$scope.openPopup = function () {
        ngDialog.open({
            template: '<div><h2>This is the popup</h2></div>',
            plain: true,
            scope: $scope
        });
    };

I also learnt how to call an external html file and assign it a new scope and a controller from the current controlle. It is done as below:

ngDialog.open({
            template: 'partials/forgotPassword.html',
            controller: 'ForgotPasswordCtrl',
            scope: $rootScope.$new()
        });
