Widgets

In this experiment, I created a toolbar and icons for each widget.

When clicked on the icon, the widget opens. It can again be closd by clicking on the icon again or by clicking on the close button of the widget.


For this experiment, I created 5 angularJS directives. Click here to view


The browser has a facility of adding new tabs. You can a new tab by douuble clicking on the browser toolbar.


The youtube widget allows to search different videos from youtube. Enter your favorite, and click enter to get result.


Youcan also change the background, by clicking on the change background icon, pasting the image URL and clicking on Go.


The main complexity of this experiment is adjusting the iframe, and textarea dynamically on resizing the widget. It is handled as below:

myApp.directive('adjustIframe', function ($timeout, $window) {
    return {
        restrict: 'A',
        link: function (scope, element) {
            scope.iframeHeight = element.prop('offsetHeight') - 76;
            angular.element($window).bind('mouseup', function () {
                scope.iframeHeight = element.prop('offsetHeight') - 76;
                scope.$apply();
            });
        }
    };
});
