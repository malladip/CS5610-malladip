For my project, I will be using MongoDB as the back end.

So the following is the database schema which I devloped for my project:


All the usernames and passwords will be saved into the Authentication collection.

var AuthentcationSchema = new mongoose.Schema({
    username: String,
    password: String
}, { collection: "Authentication" });

The ID field in the Authentication collection is used to uniquely identify each user. It is also used to get other details of users like profile info messages etc.

var UserProfileSchema = new mongoose.Schema({
