Anchor Tag

  • The below code creates a link to www.wikipedia.org. When clicked, opens in current tab

    <a href="http://en.wikipedia.org/wiki/Main_Page">Open Wiki in same tab</a>

    Open Wiki in same tab

  • Opens in new tab

    <a href="http://en.wikipedia.org/wiki/Main_Page" target="_blank">Open Wiki in new tab</a>

    Open Wiki in new tab

  • Opens in iframe

    <a href="http://en.wikipedia.org/wiki/Main_Page" target="inline">Open Wiki below </a>

    Open Wiki below

  • Go to a part in the same page

    <a href="#top">Top</a>

    Top
