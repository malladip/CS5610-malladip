Basic HTML Tags

  • The text in <b> tag is printed in bold

    <b>This text is written in bold</b>

    This text is written in bold

  • The text in <i> tag is printed in italics

    <i>This text is written in italics</i>

    This text is written in italics

  • The text in <u> tag is underlined

    <u>This text is underlined</u>

    This text is underlined

  • HTML provides with a block tags for headings. There are 6 headings tags. <h1> to <h6> They take up the entire width of the page.

    <h1>This is h1 tag</h1>
    <h2>This is h2 tag</h2>
    <h3>This is h3 tag</h3>
    <h4>This is h4 tag</h4>
    <h5>This is h5 tag</h5>
    <h6>This is h6 tag</h6>

    This is h1

    This is h2

    This is h3

    This is h4

    This is h5
    This is h6
