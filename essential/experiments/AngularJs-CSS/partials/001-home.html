Changing CSS Dynamically in AngularJS


The main aim of this experiment is to change the CSS of the page dynamically while using AngularJS

A user after loging in can change his background. It remains persistent even after loging off.

For practice purpose,I hardcoded an initial user with the username and password as below:

Username : admin
Password : admin

Here I learnt how to change the CSS of the following:

  • the navigation bar - container page CSS

    .activeStyle {  //done by simply linking CSS page to the container page
    font-size: 25px;
}

  • the background image on change of tabs - change the CSS of container page on change in container page

    if ($location.path() === '/register') {   // done by adding this code in the controller of container page
    $scope.background = { 'background-image': 'url(required url)' };
}

  • the background image of a user - change the CSS of container page on change in partial pages

    $scope.changeBg = function () {    // done by adding this code in the partial page controller
    var url = $scope.backgroundURL;
    LoginService.changeUserBG(url);
    $scope.backgroundURL = null;
};
    var changeUserBG = function (url) {   // Adding the value into a service and using the service variable in container page
    for (user in users) {
       if (users[user].username == currentUser) {
          users[user].background = url;
          currentBackground = users[user].background;
      }
    }
};

  • adding CSS in the partial page

    .contact{   //Simply by adding the link into the container page and using the class in the partial page
    font-size: 50px;
    font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
}
