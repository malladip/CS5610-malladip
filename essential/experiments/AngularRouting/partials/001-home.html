Hello angularJS Routing


Here I created a base container screen, and three partials screens(which will be injected into the container screen).

My second page is a login page, which is a basic login functionality. It displays weather the entered username and password is valid or not.

For practice purpose,I hardcoded the username and password.

Username : admin
Password : admin

    app.config(['$routeProvider',
function ($routeProvider) {
    $routeProvider.
    when('/home', {
        templateUrl: 'partials/001-home.html'
    }).
    when('/login', {
        templateUrl: 'partials/001-login.html',
        controller: 'LoginController'
    }).
    when('/contact', {
        templateUrl: 'partials/001-contact.html'
    }).
    otherwise({
        redirectTo: '/home'
    })
}
]);
