Login functionality using AngularJS


I extended my second angularJs experiment to add a register functionality.

The register option allows a user to create his own username and password.

For practice purpose,I hardcoded an initial user with the username and password as below:

Username : admin
Password : admin

In this experiment, I had to change the data format of the users.

var users = [{ username: 'admin', password: 'admin' }];

For pushing the newly added users into the users variable, push() function is used. 

var new_user = { username: username, password: password };
users.push(new_user);

As the the users data format changed, the way of cheking the username and password at login time also changes as below: 

    for (var i = 0; i < users.length; i++) {
            if (username == users[i].username && password == users[i].password) {
                currentUser = username;
                return currentUser;
            }
        }
