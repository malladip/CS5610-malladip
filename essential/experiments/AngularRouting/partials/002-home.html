Login functionality using AngularJS


I extended my first angularJs experiment to add a login functionality.

Here, user can login to see a new tab called profile and a logout option is available.

For practice purpose,I hardcoded the username and password.

Username : admin
Password : admin

In this experiment, I had to create a sevice to store the current logged in user's username.

app.factory("LoginService", function () {
var currentUser = null;
var login = function (username, password) {
    if (username == "admin" && password == "admin") {
        currentUser = username;
        return currentUser;
    }
    return null;
};
    
return {
    login: login,
    getCurrentUser: getCurrentUser
}
});

The main difficulity of this experiment was to notify the container's controller about the change in value of services's currentUser variable at the time of login. For getting the updated value, on change of the value, the below code is used: 

    $scope.$watch(
        function () {
            return LoginService.getCurrentUser();
        },
       function (currentUser) {
           $scope.currentUser = currentUser;
       }, true);
