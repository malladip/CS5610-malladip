Here I learnt how to embed a youtube video in angularJS.

We can not give the video link directly in the ng-src directive of an iframe.

We need to use the following code

$scope.trustSrc = function (src) {
  return $sce.trustAsResourceUrl(src);
}

And ng-src should be as below

ng-src="{ {trustSrc(video)} }"
