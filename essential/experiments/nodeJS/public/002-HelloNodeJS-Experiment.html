This is my first NodeJS experiment.

Here I tries to get a responce from the server on a button click.

The Http Call

app.controller("ContainerController", function ($scope,$http) {
    $scope.getResponce = function () {
        $http.get('/experiment2/getresponce')
        .success(function (responce) {
            $scope.responce = responce;
        });
    }
});

The Server side code

app.get("/experiment2/getresponce", function (req, res) {
    res.send('Hi! this is the responce from the server!! success!!');
});
