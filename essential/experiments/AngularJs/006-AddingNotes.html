In this experient, I played with angularJS and ng-model. I tried adding new notes, deleting them and copying them.

Repeat by default does not allow duplicate value. In order to overcome this, we can track by index as below:

ng-repeat="note in notes track by $index"

Adding new note:

if (!$scope.notes) {
$scope.notes = [];
}
$scope.notes.push(text);
console.log($scope.notes);

Deleting note:

$scope.notes.splice(index, 1);
        
if ($scope.notes.length == 0) {
$scope.notes = undefined;
}

Copying note:

$scope.text = note;
